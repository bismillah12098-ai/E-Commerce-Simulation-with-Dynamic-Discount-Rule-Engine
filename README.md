# ECommerceSimulation (Python / FastAPI)

## Overview
FastAPI demo of an e-commerce backend implementing design patterns:
- Factory Method (ProductFactory)
- Abstract Factory (CategoryFactory)
- Builder (CartBuilder)
- Decorator (Discounts)
- Facade (CheckoutFacade)
- Singleton (InventorySingleton)

## Requirements
- Python 3.9+
- Virtualenv recommended

## Install & Run
1. Create virtualenv:
   - Windows:
     ```
     python -m venv .venv
     .venv\Scripts\activate
     ```
   - macOS / Linux:
     ```
     python -m venv .venv
     source .venv/bin/activate
     ```
